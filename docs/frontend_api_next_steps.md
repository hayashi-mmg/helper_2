## 📋 品質保証

### **テスト実行環境の整備**

```bash
# テスト環境セットアップ
npm install
npm run test:setup

# 継続的テスト実行
npm run test:watch          # 開発時
npm run test:coverage       # カバレッジ確認
npm run test:e2e           # E2Eテスト
npm run test:e2e:headed    # ブラウザでE2Eテスト
```

### **コード品質チェック**

```bash
# 静的解析
npm run lint               # ESLint
npm run type-check         # TypeScript
npm run prettier          # コードフォーマット

# 自動修正
npm run lint:fix
npm run prettier:write
```

### **パフォーマンス監視**

```bash
# ビルド解析
npm run build:analyze      # バンドルサイズ分析
npm run lighthouse         # パフォーマンススコア
```

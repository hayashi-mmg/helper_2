# 1週間先献立作成アプリ 画面仕様書

## 1. 概要

### 1.1 システム概要
ホームヘルパー向けの献立管理システム。1週間先の献立を効率的に作成・管理し、複数品目の料理を組み合わせた実用的な食事プランを提供する。

### 1.2 対象ユーザー
- 利用者（高齢者等）
- ホームヘルパー
- ケアマネージャー

---

## 2. メイン画面構成

### 2.1 ヘッダー部
```
┌─────────────────────────────────────────────────┐
│              1週間先献立作成                        │
│              (text-3xl, 中央配置)                   │
└─────────────────────────────────────────────────┘
```

### 2.2 週選択ナビゲーション部
```
┌─────────────────────────────────────────────────┐
│ [◀ 前の週]     今週 7/13-7/19     [次の週 ▶]    │
│                                                  │
│  [今週に戻る] [前週をコピー] [クリア]              │
└─────────────────────────────────────────────────┘
```

**機能仕様:**
- 前の週/次の週ボタン: 表示週を変更
- 週表示: 「今週」「来週」「再来週」「N週間後」
- 日付範囲: MM/DD-MM/DD形式
- 今週に戻る: selectedWeekOffset = 0
- 前週をコピー: 前週の献立を現在の週にコピー
- クリア: 現在の週の献立を全削除

### 2.6 買い物リスト生成部
```
┌─────────────────────────────────────────────────┐
│ 🛒 買い物リスト生成                              │
│                                                  │
│ [📋 この献立から買い物リストを作成]               │
│                                                  │
│ (生成後に表示)                                    │
│ ┌─────────────────────────────────────────────┐ │
│ │ 買い物リスト: 7/13-7/19の週                  │ │
│ │                                              │ │
│ │ ☑ 鶏もも肉   [肉類]  600g              [除外]│ │
│ │   └ 照り焼き300g + 親子丼300g               │ │
│ │ ☑ 卵        [卵・乳] 5個               [除外]│ │
│ │   └ 目玉焼き2個 + 卵焼き2個 + 親子丼1個    │ │
│ │ ☐ しょうゆ  [調味料] 大さじ4  (在庫あり)    │ │
│ │   └ 照り焼き大さじ2 + 親子丼大さじ2        │ │
│ │ ☐ みりん    [調味料] 大さじ2  (在庫あり)    │ │
│ │                                              │ │
│ │ 合計: 15品  除外: 3品  購入対象: 12品        │ │
│ │                                              │ │
│ │ [ヘルパーに買い物を依頼する]                  │ │
│ └─────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────┘
```

**機能仕様:**
- **生成ボタン**: 週間献立が存在する場合のみ活性化
- **食材一覧**: 生成された買い物アイテムをカテゴリ別にグルーピング
- **除外トグル**: 各アイテムの除外ボタンで在庫あり/除外切り替え
  - 除外済みアイテム: 取り消し線 + 灰色テキスト + 除外理由表示
  - 「在庫あり」はパントリー由来の自動除外を示す
- **出典表示**: 各食材の下に使用レシピと内訳を小文字で表示
- **サマリー**: 合計品数・除外品数・購入対象品数を表示
- **依頼ボタン**: 除外されていないアイテムをShoppingRequestとして送信

**除外アイテムスタイル:**
```css
/* 除外済み */
line-through text-gray-400 bg-gray-50
/* 除外理由バッジ */
bg-yellow-100 text-yellow-700 text-xs
```

- 検索窓: 料理名での部分一致検索

- ジャンル・タイプバッジ
- 調理時間・難易度表示
- 材料プレビュー（最大3つ、以降は...）

## 5. データ構造仕様

### 5.1 レシピデータ構造
```javascript
{
  id: Number,           // ユニークID
  name: String,         // 料理名
  category: String,     // ジャンル（和食/洋食/中華/その他）
  type: String,         // タイプ（主菜/副菜/汁物/ご飯/その他）
  cookTime: Number,     // 調理時間（分）
  difficulty: String,   // 難易度（簡単/普通/難しい）
  ingredients: Array    // 材料リスト
}
```

### 5.2 献立データ構造
```javascript
weeklyMenus = {
  "週キー": {           // "YYYY-M-D" 形式
    "日インデックス": {  // 0-6 (月-日)
      "朝食": [recipeId1, recipeId2, ...],
      "夕食": [recipeId1, recipeId2, ...]
    }
  }
}
```

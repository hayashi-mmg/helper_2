## 4. 技術的重要ポイント

### 4.1 高齢者向け最重要原則
**「圧倒的なアクセシビリティの確保」**

- **フォント**: 基本18px、最小16px
- **コントラスト**: 4.5:1以上必須
- **タップターゲット**: 最小44px × 44px
- **認知負荷**: 1画面1タスク、シンプルナビ

### 4.2 データベース設計のポイント
- **STI方式**: ユーザー種別を`role`カラムで管理
- **中間テーブル**: 献立の複数品目組み合わせ
- **インデックス**: 1,000同時接続に対応
- **接続プーリング**: PgBouncer等必須

### 4.3 セキュリティ実装重点
- **JWT戦略**: アクセス30分 + リフレッシュ7日
- **QRコード**: 24時間有効ワンタイムトークン
- **監査ログ**: 全操作の記録・追跡
- **暗号化**: 個人情報の適切な保護

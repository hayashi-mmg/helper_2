# ヘルパー向け作業管理ダッシュボード 画面仕様書

## 1. 概要

### 1.1 システム概要
ホームヘルパーの日々の作業を効率的に管理するダッシュボード。利用者への食事準備、定期作業、特別な依頼、買い物依頼管理、およびコミュニケーション機能を一元化した作業支援システム。

### 1.2 対象ユーザー
- ホームヘルパー（主要ユーザー）
- ケアマネージャー（進捗確認）
- 利用者（間接的な情報共有）

### 1.3 画面アクセス方法
- 利用者宅到着時にアプリを起動
- 作業開始前に当日の作業内容を確認
- 作業中の進捗管理とコミュニケーション
- 作業完了時の報告

## 3. 各部詳細仕様

### 3.1 ヘッダー部
```
┌─────────────────────────────────────────────────────────┐
│ ヘルパー作業ダッシュボード           14:32               │
│ 2025年7月13日（日）                09:00-12:00          │
│                                                         │
│ 今日の進捗 ████████░░ 80%                               │
└─────────────────────────────────────────────────────────┘
```

**機能仕様:**
- **タイトル**: "ヘルパー作業ダッシュボード" (text-2xl, font-bold)
- **現在時刻**: リアルタイム更新（1分間隔）、HH:MM形式
- **作業日**: YYYY年MM月DD日（曜日）形式
- **作業時間**: 開始時刻-終了時刻
- **進捗バー**: 全タスクの完了率を視覚化、青色グラデーション

**進捗計算ロジック:**
```javascript
完了率 = (完了タスク数 / 全タスク数) × 100
全タスク = 食事準備 + 定期作業 + 特別依頼
```

```javascript
todaysTasks = {

  schedule: {

  meals: {

      }
    ],
    dinner: [...]        // 夕食の料理配列
  },
  regularTasks: [        // 定期作業配列
    {
      id: String,
      name: String,
      scheduled: Boolean,
      completed: Boolean,
      estimatedTime: Number
    }
  ],
  specialRequests: [     // 特別依頼配列

      id: Number,
```
